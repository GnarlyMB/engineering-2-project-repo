# engineering-2-project-repo
A repository to be used for OSU ENG 1182 semester project.

## Description:
This is the software that will be implemented into the alarm clock.

## Start up:
1. Clone the repository into the Matlab IDE.
2. Install the proper Arduino app (Arduino Explorer).
3. Check the pinout blueprint and read the comments 
    in the code to see what needs to be finished.

## Commiting Code: 
- Please ensure that the commit comment properly explains 
    the changes made or content added.
- Make sure that the code is properly formatted and commented.
